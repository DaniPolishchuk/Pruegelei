# 🥋 2D Fighter Game

A **pure JavaScript** 2D fighting game – no frameworks involved!  
Choose between two game modes: **Local** and **Multiplayer**.

---

## 🎮 Controls

### 👤 Local Mode

#### Player 1:
- Move / Jump: `A` `D` `W`  
- Block: `S`  
- Attacks: `1` `2` `3` `4`

#### Player 2:
- Move / Jump: `←` `→` `↑`  
- Block: `↓`  
- Attacks: `0` `9` `8` `7`

### 🌐 Multiplayer Mode
- Use **Player 1 controls** from above.

---
